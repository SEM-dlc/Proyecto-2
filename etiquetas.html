Etiquetas Fundamentales en HTML

Etiquetas de Texto

Las etiquetas de texto en HTML permiten estructurar y dar formato al contenido textual de una página web:


  1. h1 a h6: Estas etiquetas representan los encabezados, siendo `h1` el más importante y `h6` el menos importante. Los encabezados se utilizan para definir títulos y subtítulos, ayudando a estructurar el contenido y mejorando la accesibilidad y el SEO.

  2. p: Define un párrafo. Es una de las etiquetas más comunes en HTML, utilizada para agrupar bloques de texto.

  3. a: Crea un enlace o *hipervínculo* a otra página web, documento o recurso.

  4. strong y em: Estas etiquetas se utilizan para dar énfasis al texto. `strong` aplica un énfasis fuerte (normalmente se muestra en negrita), mientras que `em` aplica un énfasis leve (normalmente se muestra en cursiva).



Etiquetas de Formato

Además de las etiquetas de texto, HTML proporciona etiquetas para dar formato al contenido:


  1. br: Inserta un salto de línea. Es útil para separar líneas de texto sin iniciar un nuevo párrafo.

  2. hr: Inserta una línea horizontal que puede usarse para separar secciones de contenido.

  3. blockquote: Define una cita en bloque, usada para citar un texto extenso proveniente de otra fuente.
