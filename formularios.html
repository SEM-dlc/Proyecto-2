Formularios y Tablas en HTML

Formularios HTML

Los formularios en HTML son fundamentales para la interacción del usuario con la web. Permiten recopilar datos a través de entradas de texto, botones, casillas de verificación, y otros controles de formulario:


  • Estructura básica de un formulario: Un formulario en HTML se define con la etiqueta `form`. Dentro de esta, se colocan varios elementos de entrada como `input`, `textarea`, `button`, y `select`.



Tipos de entradas y atributos esenciales:



  • input type="text": Entrada de texto.

  • input type="password": Entrada de contraseña (oculta el texto).

  • input type="radio" y input type="checkbox": Botones de opción y casillas de verificación.

  • Atributos importantes: `name` (nombre del campo), `type` (tipo de entrada), `value` (valor por defecto), `placeholder` (texto de marcador).


Tablas HTML

Las tablas en HTML se utilizan para organizar datos en filas y columnas. Aunque su uso ha disminuido para el diseño de páginas, siguen siendo útiles para presentar datos tabulares:


  • Estructura básica de una tabla: Las tablas se crean usando la etiqueta `table`, con filas (`tr`), celdas de datos (`td`), y celdas de encabezado (`th`).

  • Atributos de formato: Las tablas pueden personalizarse con atributos como `border` (borde de la tabla), `cellpadding` (espacio dentro de las celdas), y `cellspacing` (espacio entre celdas).


HTML Semántico

HTML semántico hace referencia al uso de etiquetas que no solo definen la apariencia del contenido, sino también su significado. Esto mejora la accesibilidad, facilita el mantenimiento del código, y optimiza la interpretación de la página por parte de los motores de búsqueda:



Etiquetas semánticas comunes:


  • header: Define la cabecera de un documento o una sección.

  • nav: Representa un conjunto de enlaces de navegación.

  • section: Agrupa contenido temáticamente relacionado.

  • article: Representa un contenido independiente, como un artículo o entrada de blog.

  • footer: Define el pie de página.
