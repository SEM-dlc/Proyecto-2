¿Qué es HTML?


HTML, o "Hyper Text Markup Language", es el lenguaje estándar utilizado para crear y diseñar documentos web. Es un lenguaje de marcado, lo que significa que utiliza etiquetas (o *tags*) para definir la estructura y el contenido de una página web.


Los títulos, párrafos, enlaces, imágenes y otros componentes multimedia se definen por medio de cada etiqueta. El HTML es un lenguaje de marcado que permite organizar y presentar el contenido de manera coherente y legible para los navegadores web y los motores de búsqueda, no es un lenguaje de programación. Es esencial para el desarrollo web y sirve como base para otras tecnologías web, como JavaScript y CSS (Cascading Style Sheets).

Historia de HTML

En 1991, Tim Berners-Lee creó el HTML mientras trabajaba en el Laboratorio Europeo de Física de Partículas del CERN. HTML 1.0, la versión inicial, era muy simple y permitía la creación de documentos sencillos con enlaces y texto. HTML ha cambiado mucho desde entonces, con la introducción de nuevas versiones que han mejorado sus habilidades y su adaptabilidad:


  • HTML 2.0 (1995): Agregó funciones adicionales, como tablas y formularios.
  • HTML 3.2 (1997): Agregó soporte para hojas de estilo, scripts y applets.
  • HTML 4.01 (1999): mejoró la accesibilidad y el apoyo a los elementos multimedia.
  • XHTML 1.0: Un intento de transformar HTML en XML en el año 2000.
  • HTML5 (2014): La versión más reciente incluye mejor soporte multimedia, APIs como el almacenamiento local y nuevas etiquetas.
