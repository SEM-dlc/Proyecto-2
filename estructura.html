Estructura Básica de un documento HTML



Un documento HTML típico comienza con una declaración de tipo de documento (``) que informa al navegador sobre la versión de HTML utilizada. La estructura básica de un documento HTML incluye las siguientes secciones:

  1. `html`: La raíz del documento HTML. Todo el contenido debe estar dentro de esta etiqueta.

  2. `head`: Contiene metadatos sobre el documento, como el conjunto de caracteres (`meta charset="UTF-8"`), el título de la página (`title`), y enlaces a hojas de estilo o scripts externos.

  3. `title`: Define el título de la página que aparece en la pestaña del navegador

  4. body` : Contiene el contenido visible de la página web, como texto, imágenes, enlaces, etc.
